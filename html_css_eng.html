Web Development Course (Syllabus):

1. Introduction to HTML.
2. Introduction to CSS.
3. Integration of Bootstrap 5.
4. Creation of personal static web page.
5. Hosting of personal static web page.

Course Description


The course is intended for people of any age and does not require any type of programming background. The course is conducted online and its duration is two months. After completing the course, the student will have a static website built by herself/himself and will be able to start learning Python or JavaScript programming languages. The lecture is held once a week and lasts an hour and thirty minutes.
